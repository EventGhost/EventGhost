Orange LiveBox
===========

OrangeLiveBox is a plug-in for EventGhost (http://eventghost.org/)
This plug-in will allow you to control your box over ethernet with EventGhost.

With this plug-in you can:
- emulate remote control buttons 

Installation
============

Download the .zip file from https://github.com/flopp26/EG-OrangeLiveBox
Unzip it into the EventGhost plug-in directory

Requirements
============
    
In EventGhost:
    Configure the box device IP address or hostname
    The port should not need any configuration.

Compatibility
=============

LiveBox Play IHD92
